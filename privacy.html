🔒 隐私政策

最后更新日期：2026年1月1日 · 生效日期：2026年1月1日

重要提示：Alight 是一款本地运行的 Mac 清理工具，我们不会收集、上传或存储您的任何个人数据。您的隐私完全由您自己掌控。

1. 引言

欢迎使用 Alight（以下简称"本软件"或"我们"）。我们深知个人隐私的重要性，并致力于保护您的个人信息安全。本隐私政策旨在向您说明我们如何处理与您使用本软件相关的信息。

请您在使用本软件前仔细阅读本隐私政策。使用本软件即表示您同意本隐私政策的条款。

2. 信息收集

2.1 我们不收集的信息

本软件是一款完全本地运行的桌面应用程序，我们郑重声明：

  • 不收集个人身份信息：我们不会收集您的姓名、电子邮件地址、电话号码等任何个人身份信息。
  • 不收集设备信息：我们不会收集您的设备型号、序列号、MAC 地址等硬件信息。
  • 不收集使用数据：我们不会记录您的操作行为、使用习惯、清理记录等使用数据。
  • 不收集文件信息：我们不会上传或存储您电脑上的任何文件信息、路径信息或文件内容。
  • 不使用 Cookie：本软件不使用任何 Cookie 或类似技术追踪您的活动。

2.2 本地存储的信息

为了提供更好的用户体验，本软件会在您的设备本地存储以下信息：

  • 用户偏好设置（如主题、语言等）
  • 清理规则配置
  • 扫描历史记录（仅用于本地展示）

以上信息仅存储在您的本地设备，不会上传至任何服务器。您可以随时通过清除应用数据来删除这些信息。

3. 数据处理

3.1 本地处理原则

本软件的所有核心功能均在您的设备本地完成，包括但不限于：

  • 文件扫描与分析
  • 缓存清理
  • 应用卸载
  • 启动项管理
  • 进程监控

3.2 权限使用说明

本软件运行需要以下系统权限，这些权限仅用于实现软件功能，不会用于其他目的：

  • 文件系统访问权限：用于扫描和清理文件
  • 辅助功能权限：用于获取应用列表和管理进程
  • 全盘访问权限：用于深度扫描系统缓存和日志

您可以在 macOS 系统设置中随时撤销这些权限，但可能会影响软件的某些功能。

4. 网络通信

零网络依赖：本软件的核心功能完全离线可用，不需要网络连接即可正常使用。

4.1 版本检查（可选）

当您启用"自动检查更新"功能时，本软件会向 GitHub Releases API 发送请求以检查新版本。此请求：

  • 仅获取最新版本号信息
  • 不发送任何个人识别信息
  • 您可以在设置中关闭此功能

4.2 下载更新

当您选择下载更新时，软件将从 GitHub 或我们指定的服务器下载安装包。下载过程遵循 GitHub 的隐私政策。

5. 第三方服务

本软件不集成任何第三方分析工具、广告SDK或数据收集服务。我们不与任何第三方共享您的数据，因为我们根本不收集数据。

6. 数据安全

由于本软件不收集或上传任何数据，数据安全完全由您自己掌控。我们建议您：

  • 定期备份重要文件
  • 使用强密码保护您的 Mac
  • 保持 macOS 系统为最新版本
  • 仅从官方渠道下载本软件

7. 儿童隐私

本软件不针对 13 周岁以下儿童。我们不会故意收集儿童的个人信息。由于我们实际上不收集任何数据，因此不存在儿童隐私风险。

8. 国际用户

本软件面向全球用户。由于所有数据均在本地处理，不涉及数据跨境传输，因此符合中国《个人信息保护法》、欧盟 GDPR 等全球主要隐私法规的要求。

9. 隐私政策的变更

我们可能会不时更新本隐私政策。更新后的政策将在本页面发布，并在页面顶部显示新的生效日期。重大变更将通过软件内通知或官网公告的形式告知您。

建议您定期查阅本政策，以了解我们如何保护您的隐私。

10. 您的权利

根据相关法律法规，您享有以下权利：

  • 知情权：了解软件如何处理您的数据
  • 删除权：删除本地存储的所有应用数据
  • 拒绝权：关闭自动更新检查功能
  • 卸载权：随时卸载本软件

11. 免责声明

重要提示：本软件仅作为工具辅助您清理和管理 Mac 系统。使用本软件进行的任何清理操作均由您自行决定并确认。对于因使用或误用本软件导致的任何数据丢失、系统故障或其他损失，我们不承担任何责任。
